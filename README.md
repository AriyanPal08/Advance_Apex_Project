# BFSI Compliance Automation Pipeline

## Project Description
This project builds a data pipeline that ingests raw complaint records from the CFPB Consumer Complaint Database, resolves data quality issues, standardizes classification fields, encodes compliance status, and produces a clean, analysis-ready compliance dataset.

## File Structure
- `filter_data.py`: Prepares the working dataset by reading the original CSV in chunks, filtering to years 2022-2024, and randomly sampling 350,000 rows.
- `apex_project_FINAL.ipynb`: The primary Jupyter Notebook for data inspection, cleaning, encoding, and analysis. Runs top-to-bottom without errors.
- `Project_Report.docx`: A detailed technical report outlining the dataset, pipeline decisions, before/after data quality metrics, key findings, and conclusions.
- `GenAI_Prompt_Log.docx`: A log of prompts used during the development of this project for guidance and troubleshooting.
- `cfpb_raw_2022_2024.csv`: The intermediate filtered dataset (generated by `filter_data.py`).
- `clean_compliance_data.csv`: The final, clean dataset outputted by the notebook.

## How to Run
1. Ensure the raw `complaints.csv` dataset is present in the working directory.
2. Run `python filter_data.py` to generate the 2022-2024 dataset slice.
3. Open `apex_project_FINAL.ipynb` and choose "Restart & Run All" to execute the pipeline and generate the final analysis.
